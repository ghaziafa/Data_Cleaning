# Data_Cleaning_PublicHealth-_Dataset
Dataset: Adult Diet, Physical Activity, and Weight Status (CDC BRFSS) Goal: Transform raw public health survey data into a clean, analysis-ready format.
This project walks through a full real-world data cleaning pipeline using a messy public dataset.
The dataset contains:
--Mixed data types
--Non-numeric symbols
--Text labels mixed with numbers
--Extra spaces and inconsistent formatting
--Missing values
--Potential outliers
--Duplicate rows
